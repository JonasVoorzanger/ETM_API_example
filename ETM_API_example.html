The Energy Transition Model API

Introduction

The Energy Transition Model (ETM), created by Quintel Intelligence, has a freely available Application Programming Interface (API). This API enables developers to use the engine (or 'calculator') behind the ETM within an external application.

The example below can be used as a starting point to create your own web application with ETM interactivity. To do this, basic knowledge about HTML, Javascript and jQuery is required (excellent tutorials are freely available on Codecademy.com). The example also uses functionality from within the D3.js and Highcharts.js libraries.

The HTML of this page and the required required Javascript files can be found on this Github repository. This PDF file shows a graphical overview of the different steps involved in initializing and updating a ETM scenario through the API.
